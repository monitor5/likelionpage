FAQ

자주 묻는 질문

개발이 처음이어도 지원 가능한가요?

가능합니다. 기본 트랙과 프로젝트 트랙을 함께 운영해, 입문자도 단계적으로 실력을 키울 수 있도록 돕습니다.

어떤 스택으로 프로젝트를 진행하나요?

팀 목표에 따라 선택하지만, 주로 React, Node.js, Python, Cloudflare, GitHub 기반 협업을 많이 사용합니다.

활동 시간은 어느 정도 필요한가요?

주 1회 정기 세션과 팀별 추가 협업 시간이 필요합니다. 실제 서비스 완성을 목표로 운영됩니다.

이번 학기, 직접 만들어볼 차례입니다

개발 많이된다에서 팀과 함께 문제를 해결하고, 결과물로 성장하세요.
